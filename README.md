# LiveAlbumARtwork

**LiveAlbumARtwork** es una aplicación móvil de realidad aumentada en proceso de refactorización, nacida originalmente como un prototipo en Unity + Vuforia para transformar portadas de álbumes musicales en experiencias AR interactivas.

La versión anterior demostró el concepto central: apuntar la cámara de un dispositivo móvil a una portada, reconocerla mediante **Vuforia Cloud Recognition** y desplegar una escena AR personalizada con visuales 3D, texto dinámico, audio preview y contenido contextual.  
Este repositorio representa la siguiente etapa: una reconstrucción más limpia, escalable y mantenible, pensada para un flujo moderno de AR móvil. 🚀

## Estado actual

Este proyecto está **en construcción**.  
Se encuentra en proceso de refactorización desde un prototipo anterior hacia una versión más estructurada, modular y sostenible.

## Qué hace la aplicación

- Reconoce portadas de álbumes mediante Vuforia
- Carga metadatos y contenido multimedia de forma dinámica
- Muestra una presentación AR con un case de CD y un disco 3D
- Reproduce previews de audio y contenido de video relacionado
- Presenta información del álbum y del track actual en tiempo real
- Soporta interacciones táctiles y futuras extensiones
- Incluye avisos legales y créditos de autoría

## Enfoque de la refactorización 🛠️

Esta nueva versión se está construyendo con énfasis en:

- una estructura más limpia del proyecto Unity
- un modelo formal de datos y contenido
- soporte futuro para distribución remota de assets
- una mejor experiencia de usuario en móviles
- una arquitectura más mantenible para desarrollo unipersonal

## Notas del repositorio

- La versión anterior del proyecto se conserva solo como referencia técnica.
- Los assets del prototipo previo no se importan en bloque a esta nueva base.
- El proyecto parte desde una plantilla moderna de AR móvil en Unity e integra Vuforia para el reconocimiento de portadas.
- Los nuevos assets, modelos y contenidos se irán incorporando progresivamente conforme avance la refactorización.

## Aviso legal y de contenido ⚖️

Este proyecto es una aplicación personal y educativa, centrada en el descubrimiento musical y la presentación inmersiva.

Todo el contenido audiovisual de los álbumes —incluyendo portadas, ilustraciones, previews musicales, videos y materiales relacionados— pertenece a sus respectivos titulares de derechos.  
La aplicación se desarrolla como una capa de presentación y descubrimiento, **no** como una reivindicación de propiedad sobre materiales de terceros.

## Repositorio público, pero no dominio público 📌

Este repositorio es público para fines de exhibición, documentación y portafolio.  
Eso significa que:

- el código, la estructura del proyecto y la documentación pueden consultarse públicamente;
- el repositorio sirve como evidencia de trabajo, aprendizaje y evolución técnica;
- su visibilidad pública **no** implica cesión de derechos de autor;
- el proyecto **no** es de dominio público;
- el contenido del repositorio no debe interpretarse como una autorización general para copiar, redistribuir, relicenciar o reutilizar el material sin el permiso correspondiente.

En otras palabras: el hecho de que el repositorio sea visible no significa que el proyecto esté liberado al dominio público ni que sus componentes puedan ser usados sin restricciones.  
Su propósito principal es mostrar el proceso de desarrollo, la calidad técnica del trabajo y su evolución como parte de un portafolio profesional.
